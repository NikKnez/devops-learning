# Month 2 Learning Materials

## Week 1-2: Networking Fundamentals

### What to learn:

-       IP addresses (IPv4, IPv6), subnets, CIDR notation
-       DNS (how domain names work)
-	Ports and protocols (HTTP/80, HTTPS/443, SSH/22, etc.)
-	HTTP/HTTPS basics (requests, responses, headers, status codes)
-	Firewalls and security groups concepts
-	Load balancers (concept and purpose)
-	SSH tunneling and port forwarding
-       Basic troubleshooting: ping, curl, wget, netstat, traceroute, nslookup, dig


### Free Resources:

1.	Networking for Web Developers (Udacity) - https://www.udacity.com/course/networking-for-web-developers--ud256
2.	Professor Messer Network+ (YouTube) - https://www.youtube.com/playlist?list=PLG49S3nxzAnmpdmX7RoTOyuNJQAb-r-gd
3.	HTTP Crash Course - https://www.youtube.com/watch?v=iYM2zFP3Zn0


### Weekend Project (Week 6):

-	Use curl to interact with public APIs (GitHub API, weather API)
-	Understand request/response cycle
-	Practice reading HTTP headers and status codes


## Week 3-4: Python Scripting for Automation

### What to learn:

-	Basic syntax, variables, data types (strings, integers, lists, dictionaries)
-	Functions and loops (for, while)
-	File operations (read, write, append)
-	Working with JSON (common in APIs and configs)
-	Making HTTP requests (requests library)
-	Basic error handling (try/except)
-       Command-line arguments (sys.argv)


### Free Resources:

1.	Python for Everybody - https://www.py4e.com/ 
2.	Automate the Boring Stuff with Python - https://automatetheboringstuff.com/ 
3.	Real Python Tutorials - https://realpython.com/start-here/ 


### Weekend Project (Week 8): Write Python scripts to:

-	Backup files automatically
-	Parse log files and extract errors
-	Make API calls and process JSON responses
-       Automate file organization
-       Send email notifications (using smtplib)
