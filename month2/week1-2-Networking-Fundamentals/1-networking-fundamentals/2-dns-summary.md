# DNS (Domain Name System)


## Concepts Learned

### DNS Basics
- Domain to IP translation
- DNS hierarchy (root, TLD, domain, subdomain)
- Record types (A, AAAA, CNAME, MX, TXT, NS)
- Query resolution process

### DNS Commands
- nslookup: Simple DNS lookup
- dig: Detailed DNS information
- dig +trace: Follow DNS path
- dig -x: Reverse lookup
- dig @server: Query specific DNS server

### DevOps Applications
- Blue-green deployments
- Service discovery
- Multi-region routing
- Load balancer configuration
- Internal vs external DNS
