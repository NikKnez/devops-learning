## Process Utilization


## Key Concepts

### Process Tracking
- top/htop - Real-time monitoring
- lsof - Open files and processes
- fuser - Find processes using files
- Threads vs Processes

### Resource Monitoring
- CPU: top, mpstat, vmstat
- I/O: iostat, iotop
- Memory: free, vmstat
- All: top, htop, sar

### Key Metrics
- Load average (1, 5, 15 min)
- CPU: us, sy, id, wa
- Memory: available (not free!)
- I/O: %util, await, throughput

### Automation
- Cron for scheduled tasks
- watch for continuous monitoring
- sar for historical data

## Commands Learned
- `top` - process monitor
- `htop` - better top
- `lsof` - open files
- `fuser` - file users
- `iostat` - I/O stats
- `free -h` - memory
- `vmstat` - system stats
- `crontab -e` - schedule jobs
