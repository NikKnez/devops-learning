```
### Task 3.2: Basic Emacs Commands
```
# Inside emacs:
# Ctrl+x Ctrl+f - open file
# Ctrl+x Ctrl+s - save
# Ctrl+x Ctrl+c - exit
# Ctrl+g - cancel command

# Basic editing:
# Just type normally to insert text
# Ctrl+k - kill (cut) line
# Ctrl+y - yank (paste)
# Ctrl+_ - undo

# Navigation:
# Ctrl+f - forward one character
# Ctrl+b - back one character
# Ctrl+n - next line
# Ctrl+p - previous line
# Ctrl+a - beginning of line
# Ctrl+e - end of line

# Practice: Type text, save with Ctrl+x Ctrl+s, quit with Ctrl+x Ctrl+c
